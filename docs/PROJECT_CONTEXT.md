# Project Context

## Product

Raven Manor — готическая match-3 игра в духе Gardenscapes.

Игрок проходит match-3 уровни, получает звёзды и открывает комнаты поместья. Каждая комната продвигает сюжет и визуальное восстановление дома.

## Vertical Slice

Текущая версия должна доказывать:

- match-3 core loop понятен и работает;
- звёзды связывают уровень с метаигрой;
- комнаты открываются по прогрессу;
- сюжет создаёт мотивацию продолжать;
- проект можно расширять без переписывания основы.

## Целевая платформа

Сейчас: браузер, вертикальный mobile-first интерфейс.

Позже возможны:
- PWA;
- мобильная упаковка;
- перенос или переиспользование логики в Unity.

## Не входит в текущий scope

- монетизация;
- сервер;
- аккаунты;
- полноценный редактор уровней;
- live ops;
- финальный арт;
- сложные бустеры;
- процедурная генерация кампании.
