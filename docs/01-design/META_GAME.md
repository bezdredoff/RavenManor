# Meta Game

Игрок получает звёзды за уровни и использует прогресс для открытия комнат.

Следующий этап развития:
- каждая комната имеет 3–5 restoration tasks;
- каждая задача стоит звёзды;
- выполнение меняет визуальное состояние комнаты;
- завершение комнаты открывает сюжетный эпизод.

## Current Prototype Implementation

- каждая из пяти комнат имеет три последовательные задачи;
- звёзды, потраченные на задачи, уменьшают доступный баланс;
- пороги комнат и уровней используют все заработанные звёзды;
- визуальная смена состояния комнаты будет добавлена в FEATURE-014;
- явное разделение earned/spent stars в save-модели будет добавлено в FEATURE-015.
