# Core Loop

Level Select → Match-3 Level → Stars → Manor → Unlock/Restore → Story → Next Level

## Session Goal
За одну короткую сессию игрок должен пройти 1–3 уровня и увидеть хотя бы одно мета-награду или сюжетный шаг.

## Failure
Проигрыш не отнимает прогресс. Игрок получает понятное объяснение и быстрый retry.
