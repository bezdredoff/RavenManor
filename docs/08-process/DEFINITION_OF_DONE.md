# Definition of Done

- acceptance criteria выполнены;
- `npm run build` проходит;
- релевантные автоматические тесты проходят;
- нет TypeScript ошибок;
- основной сценарий проверен вручную;
- нет изменений вне scope;
- документация обновлена;
- новый контракт понятен следующему AI;
- известные ограничения зафиксированы.
- Home и Match-3 проверены на 320 × 568 без document-scroll;
- touch, keyboard, safe-area и contained-scroll проверены;
- аудио не запускается до первого жеста и не блокирует игру при отсутствии Web Audio;
- reduced motion и asset fallback проверены;
- public-test checklist обновлён для релизной ветки.
