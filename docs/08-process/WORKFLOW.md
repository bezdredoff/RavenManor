# Development Workflow

1. Выбрать задачу.
2. Проверить Definition of Ready.
3. Создать копию задачи в `tasks/in-progress`.
4. Собрать context pack.
5. Передать задачу AI-исполнителю.
6. Проверить diff.
7. Запустить build.
8. Проверить вручную в StackBlitz.
9. Провести review.
10. Переместить задачу в done.
11. Сделать commit.
