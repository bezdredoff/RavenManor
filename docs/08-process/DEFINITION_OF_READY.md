# Definition of Ready

Задача готова к разработке, если:

- понятна пользовательская или техническая ценность;
- определён scope;
- перечислено out of scope;
- есть acceptance criteria;
- известны зависимости;
- указаны вероятные файлы;
- задача достаточно мала для одной итерации AI;
- нет нерешённого архитектурного вопроса.
